# Contributing to aknevrnky/perfect-panel

Thanks for your interest in contributing to Perfect Panel Api! This guide provides information on how to contribute to the project.

## Reporting Bugs

Please use the issue tracker to report any bugs.

## Suggesting Enhancements

Please use the issue tracker to suggest any enhancements.

## Your First Code Contribution

Unsure where to begin contributing to aknevrnky/perfect-panel? You can start by looking through the 'good first issue' label issues.

## Pull Requests

Please follow these steps for sending us your work:

1. Fork the repo.
2. Create a branch (`git checkout -b my-branch`).
3. Commit your changes (`git commit -am 'Add some feature'`).
4. Push to the branch (`git push origin my-branch`).
5. Create a new Pull Request.

Please make sure to include any necessary documentation in the PR and ensure that your code passes all tests before submitting it.
